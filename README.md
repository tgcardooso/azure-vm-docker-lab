# azure-vm-docker-lab

# 🚀 Azure VM + Docker Lab

Este projeto demonstra a criação de uma VM no Microsoft Azure e o deploy de uma aplicação utilizando Docker.

## ☁️ Tecnologias

- Azure
- Ubuntu Linux
- Docker
- Nginx

## 📌 Etapas

1. Criação da VM no Azure
2. Conexão via SSH
3. Instalação do Docker
4. Download da imagem Nginx
5. Execução do container
6. Acesso via navegador

## 🐳 Comandos

```bash
docker pull nginx
docker run -d -p 80:80 nginx
